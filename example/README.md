# Example

Ready-to-run terminal plotting demo.

From repo root:

```bash
bash example/run_example.sh
```

This runs:

- an ASCII Manhattan plot
- an ASCII QQ plot

using:

- default: `/home/yunye/work/gwaslab/examples/0_sample_data/t2d_bbj.txt.gz`
- optional override: set `INPUT=/path/to/sumstats.tsv` before running
